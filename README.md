# MetaLens - File Metadata Analyzer

## 📌 Overview

MetaLens is a modern web-based File Metadata Analyzer that allows users to upload Images and PDF files and instantly inspect their metadata. The application extracts important file information such as file size, MIME type, dimensions, modification date, EXIF camera details, and privacy-related metadata.

Designed with a premium glassmorphism interface, MetaLens helps users understand the hidden information stored inside files while identifying potential privacy risks such as GPS location exposure and camera information leakage.

---

## ✨ Features

### 📷 Image Metadata Analysis

* Upload JPG, JPEG, PNG, and other image formats
* View image preview
* Display image dimensions
* Display file size and MIME type
* Extract EXIF metadata
* Camera make and model detection
* Capture date detection
* GPS metadata detection

### 📄 PDF Analysis

* Upload PDF documents
* Display PDF file information
* Show file metadata summary

### 🛡 Privacy Scanner

* Detect exposed camera information
* Detect capture date metadata
* Detect GPS location metadata
* Generate privacy risk score
* Visual privacy assessment indicators

### 🎨 Premium User Interface

* Modern glassmorphism design
* Responsive layout
* Drag and drop upload support
* Smooth animations and hover effects
* Mobile-friendly interface
